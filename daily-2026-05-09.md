## 5. z-lab/dflash

**分类:** ⚡ 推理加速 | ⭐ 379 | 📈 +379 | [🔗 跳转](https://github.com/z-lab/dflash)

**是什么:** Block Diffusion 投机解码方案，用于加速 LLM 推理。支持 Qwen3、Gemma-4、Llama-3.1、DeepSeek-V4 等主流模型。

**为什么火:** 投机解码（Speculative Decoding）是当前 LLM 推理加速最热门的方向之一。DFlash 用 Block Diffusion 做 draft model，比传统 draft model 更高效。直接降低了推理成本。

**技术亮点:** 一个 draft model 适配多个 target model，不需要为每个大模型单独训练 draft model。这种 "draft 模型通用化" 的思路很有价值。

**💡 产品机会:**
- **推理优化即服务** - 做成推理加速中间件，用户不改代码，接入即提速降本
- **边缘部署加速** - 面向手机/端侧设备的推理加速 SDK
